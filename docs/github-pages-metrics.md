# GitHub Pages — Metrics Dashboard

To serve the growth metrics dashboard from this repository:

1. Open the repo on GitHub → **Settings** → **Pages**.
2. Under **Source**, choose **Deploy from a branch**.
3. **Branch:** `main` (or your default branch).
4. **Folder:** `/metrics`.
5. Save. GitHub will build and serve the contents of the `metrics/` folder.

The dashboard will be available at:

**https://&lt;username&gt;.github.io/&lt;repo&gt;/**

For this repo: **https://tailwind-architect.github.io/tailwind-architect/**

No Jekyll or build step is required; the files in `metrics/` are static HTML, CSS, and JS.
